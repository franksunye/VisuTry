# Try-On 模块 - 快速开始指南

## 🚀 快速访问

### 列表页面
- **URL**: `https://visutry.com/admin/try-on`
- **功能**: 查看所有用户的试戴活动
- **权限**: 需要管理员权限

### 详情页面
- **URL**: `https://visutry.com/admin/try-on/[taskId]`
- **功能**: 查看单个试戴任务的详细信息和图片

## 📊 列表页面功能

### 表格列
| 列 | 说明 |
|----|------|
| User | 用户邮箱和名称 |
| Time | 相对时间（如"5分钟前"） |
| Status | 任务状态（带颜色） |
| Action | 查看详情链接 |

### 状态颜色
- 🟢 **COMPLETED**: 绿色 - 试戴成功
- 🔴 **FAILED**: 红色 - 试戴失败
- 🟡 **PROCESSING**: 黄色 - 处理中
- 🔵 **PENDING**: 蓝色 - 待处理

### 分页
- 每页显示 10 条记录
- 使用 Previous/Next 按钮导航
- 显示当前页码和总页数

## 🔍 详情页面内容

### 用户信息卡片
- 邮箱
- 名称
- 用户创建时间

### 任务信息卡片
- 状态（带颜色编码）
- 创建时间
- 更新时间
- 错误信息（如果失败）

### 图片展示
三列布局显示：
1. **用户原始图片** - 用户上传的人脸照片
2. **眼镜图片** - 选择的眼镜框架
3. **试戴结果图片** - AI 生成的试戴结果

## 🔐 权限要求

- 必须以管理员身份登录
- 使用 NextAuth.js 进行身份验证
- 未授权用户将被重定向到登录页

## 📱 API 端点

### 获取列表
```bash
GET /api/admin/try-on?page=1&limit=10
```

**响应**:
```json
{
  "success": true,
  "data": {
    "tasks": [...],
    "pagination": {
      "page": 1,
      "limit": 10,
      "total": 100,
      "totalPages": 10
    }
  }
}
```

### 获取详情
```bash
GET /api/admin/try-on/[taskId]
```

## 💡 使用技巧

1. **快速查看最新活动**: 列表默认按时间倒序排列，最新的活动在最上面
2. **查看失败原因**: 点击失败的任务查看详情，错误信息会显示在任务信息卡片中
3. **图片预览**: 点击图片可以查看完整尺寸
4. **返回列表**: 在详情页点击"Back to List"按钮快速返回

## 🎯 常见场景

### 场景 1: 查看用户最近的试戴
1. 进入 `/admin/try-on`
2. 在列表中找到用户
3. 点击"View Details"查看详情

### 场景 2: 检查失败的试戴
1. 进入 `/admin/try-on`
2. 查找状态为 FAILED 的任务
3. 点击查看详情，查看错误信息

### 场景 3: 浏览所有试戴活动
1. 进入 `/admin/try-on`
2. 使用 Previous/Next 按钮分页浏览
3. 点击任何任务查看详情

## 📈 性能指标

- 列表页面加载时间: < 500ms
- 详情页面加载时间: < 300ms
- API 响应时间: < 200ms

## 🆘 故障排除

### 问题: 看不到 Try-On 菜单
**解决**: 确保你已以管理员身份登录

### 问题: 列表为空
**解决**: 可能还没有用户进行试戴操作，请稍后再试

### 问题: 图片无法加载
**解决**: 检查网络连接，或者试戴任务可能还在处理中
