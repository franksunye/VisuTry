## 🎯 核心策略

### 页面类型（4 种）

```
品牌/型号页 (400-500)
  ↓
脸型风格页 (100-150)
  ↓
类别页 (30-50)
  ↓
品牌页 (50-100)
```

### 内部链接

```
品牌页 → 型号页 → 脸型页 → 类别页
  ↓        ↓        ↓        ↓
相似推荐  推荐脸型  推荐品牌  热门款式
```

### SEO 优化

- ✅ 独立 meta title/description
- ✅ 结构化数据（Product Schema）
- ✅ Canonical 标签
- ✅ 动态 sitemap
- ✅ 内部链接
- ✅ 试戴 CTA
