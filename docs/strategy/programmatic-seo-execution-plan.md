## 🎯 核心策略

### 页面类型（4 种）

| 类型 | URL 示例 | 数量 | 优先级 |
|------|---------|------|--------|
| **品牌/型号页** | `/try/rayban-rx5121` | 400-500 | P0 |
| **脸型风格页** | `/style/round-face-glasses` | 100-150 | P1 |
| **类别页** | `/category/prescription-glasses` | 30-50 | P2 |
| **品牌页** | `/brand/rayban` | 50-100 | P2 |

### 内部链接策略

```
品牌页 → 型号页 → 脸型页 → 类别页
  ↓        ↓        ↓        ↓
相似推荐  推荐脸型  推荐品牌  热门款式
```
