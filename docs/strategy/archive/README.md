# VisuTry Strategy Archive

This folder is reserved for strategy documents that are historical, superseded, or no longer first-priority execution guidance.

The current top-level commercial source of truth is:

- `docs/strategy/commercial-strategy.md`

The current SEO/GEO product architecture source of truth is:

- `docs/strategy/seo/2026-06-30-product-architecture-seo-geo-sync.md`

## Archive criteria

Move a document here when one or more of the following is true:

1. The time window has expired.
2. A newer source-of-truth document explicitly supersedes it.
3. Its primary recommendation conflicts with the current commercial strategy.
4. It is useful as historical context but should not guide current execution.

## Rule of thumb

- Non-dated documents define current source-of-truth strategy.
- Dated documents record evidence, research, operating plans, or historical decisions.
- Active dated plans must declare their status clearly at the top.
- Archived documents should keep their original content intact and add an archive notice at the top when practical.
